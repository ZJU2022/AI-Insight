# AI-Insight: AI时代全栈开发学习路径

## 项目概述

AI-Insight 是一个面向中文开发者的系统化 AI 时代全栈学习与实践平台。我们正在将分散的示例、教程和经验沉淀为结构化的开源项目，帮助每一位学习者从“理解”逐步走向“构建”和“交付”。

项目当前聚焦于四个维度：
- **认知体系**：梳理大模型原理、行业洞察、学习路径
- **技术栈**：提供 RAG、Agent、微调部署等实战代码
- **工程实践**：沉淀可复用的项目模板、工具链与测试
- **社区共建**：鼓励贡献真实案例、经验分享与资源整理

> 想快速了解我们的建设蓝图？请查看 `docs/PROJECT_OVERVIEW.md`。

## 快速开始

1. 克隆仓库并创建虚拟环境（推荐 Python 3.10+）
   ```bash
   git clone https://github.com/<your-org>/AI-Insight.git
   cd AI-Insight
   python -m venv .venv && source .venv/bin/activate
   pip install -e .[dev]
   ```
2. 配置环境变量（以 OpenAI 为例）
   ```bash
   # 方式一：写入 .env
   echo "OPENAI_API_KEY=your_key" >> .env
   echo "SERPAPI_API_KEY=your_key" >> .env  # 用于章节二的搜索工具

   # 方式二：创建 Agent/config/local.py
   cat <<'PY' > Agent/config/local.py
   OPENAI_API_KEY = "your_key"
   SERPAPI_API_KEY = "your_key"
   PY
   ```
3. 运行示例
   ```bash
   python examples/react_agent_basic.py
   python Agent/第二章/react_agent.py
   python Agent/第三章/openai_test_01.py
   python Agent/第七章/babyagi_example.py
   ```

> 提示：我们正在将示例脚本逐步迁移到统一的 `src/ai_insight/` 包和 `examples/` 目录，欢迎关注后续更新。
